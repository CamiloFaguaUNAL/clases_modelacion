2) Introduction

Waldo Tobler made famous his first law of geography, which states that everything is related to everything else, but near things are more related than distant things (Tobler, 1970). Nowadays, data that obey the first law of geography are said to be spatially autocorrelated. The prefix auto means self. Thus, to say a feature is spatially autocorrelated means etymologically that its attribute values are correlated with attribute values of the same feature at nearby locations (Bandyopadhyay, 2012). Anselin and Bera (1998) provide a concise verbal definition: spatial autocorrelation can be loosely defined as the coincidence of value similarity with locational similarity.; For instance, nearby locations would probably tend to have similar mean annual precipitation levels.

Anselin and Bera (1998) provided a more formal definition: a nonzero spatial autocorrelation exists between attributes of a feature defined at locations i and j if the covariance between feature attribute values at those points is nonzero. If this covariance is positive (i.e., if data with attribute values above the mean tend to be near other data with values above the mean), then we say there is positive spatial autocorrelation; if the converse is true, then we say there is negative spatial autocorrelation. Positive autocorrelation is much more common in nature, but negative autocorrelation does exist, for example, in the case of conspecific allelopathy, the tendency of some plants to inhibit the nearby growth of other plants of the same species (Rice, 1984).

NOTE: Correlation is defined as the extent to which two variables are linearly related. Correlation tells us the strength and the direction of this relationship. Covariance is a quantitative measure of the degree to which the deviation of one variable (X) from its mean is related to the deviation of another variable (Y) from its mean. To simplify, covariance measures the joint variability of two random variables. You can find the equations and other details about the difference between correlation and Covariance in this link.

Spatial data (sampling) discussed in this tutorial consist of sets of attribute values Y(x, y) with the coordinates x and y describing the location of one or more attribute values (or response variable(s)). The locations (x, y) are fixed and not random, but the attribute values (Y) are assumed to have a random component. Errors in measuring the location of the phenomenon are incorporated into the uncertainty in the value of the phenomenon at each location. The coordinates (x,y) themselves are assumed to be measured without error.

Since one of the main objectives of spatial modelling is to predict a response variable on the terrain, we have different ways to deal with the prediction of this variable using the principles of spatial autocorrelation; we call these methods spatial interpolations. We can find two groups of spatial interpolations: (1) Methods which are based only on the mathematical calculations of spatial information (i.e., coordinates data) and do not involve spatial changes of data variance. (2) Methods that include in the mathematical calculations spatial changes of data variance.

3) Methods based only on spatial information.

The predictions of these methods can be represented as the weighted averages of sampled data and the same general prediction formula can be presented for them:

\[ y(x_0) = {\sum_{i=1}^{n}{lamda_i * y(x_i)}} \] where \(y(x_0)\) is a predicted value of an attribute at the location of interest \(x_0\), \(y(x_i)\) is observed value at the sampled location \(x_i\), \(lamda_i\) the weight assigned to the sample at \(x_i\), and \(n\) represents the number of samples used to generate the predictions. It is important to highlight term attribute is often used in interpolations and is also called primary variable and is equivalent to response variable or dependent variable. The types of methods based only on spatial information change the statistical way to determine the weights (\(lamda_i\)) (Li, 2022).

3.2) Inverse Distance Weighted (IDW).

This interpolation method,IDW, predicts the value of an attribute at an unsampled location using a linear combination of values of samples weighted by an inverse function of the distance from the location of interest to the sampled locations. The assumption is that sampled locations closer to the unsampled location are more similar to it than those further away in their values (Li, 2022).

\[ lamda_i = (1/d_i^{n}) / {\sum_{i=1}^{n}{1/d_i^{p}}} \]

where \(d_i\) is the distance between \(x_0\) and \(x_i\), p is a distance weighting power parameter, and n represents the number of samples to be used. The main factor affecting predictive accuracy of IDW is the value of the distance weighting power parameter (p). The number of samples used is also an important factor for the accuracy. Weights diminish as the distance increases, especially when the value of p increases, so nearby samples have a heavier weight and thus have more influence on the predictions.

To run IDW, we are going to install and load the library gstat for Spatial and Spatio-Temporal Geostatistical Modelling, Prediction and Simulation. The function idw() does the statistical test. The output variable is called var1.pred, and the var1.var values are NA because inverse distance does not provide prediction error variances.

4) Methods based on spatial changes of data variance.

4.1) The Variogram.

In geostatistics the spatial correlation is modeled by the variogram (also caleed semivariogram) instead of a correlogram or covariogram. Variogram plots the semivariance (semi) of the attribute (Y or response variable) as function of distance, where the semivariance is defined as a half the variance of the differences between all possible points spaced a constant distance (Li, 2022).

\[ semi = (x_i, x_0) = semi(h) = var [Y(x_i) - Y(x_0)]/2 \]

where Y is the attribute, h is the distance between data point Xi and Xo, and semi(h) is the semivariogram.

Variogram states that the variance of Y is constant, and that spatial correlation of Y does not depend on location, but only on separation distance h. The variogram allows to plot all possible pairs of Y and their estimated correlations. If we further assume isotropy (direction independence of semivariance), we can plot the absolute value of the distances. This estimate is called the sample variogram.

First, we are going to plot the variogram cloud (the plot of all possible squared differences of observation pairs against their separation distance).

plot(variogram(log(zinc) ~ 1, meuse, cloud = TRUE))

We can smooth the variation of the variogram cloud by averaging the cloud values over distance intervals to provides an estimate of semivariance. The ~ 1 defines a single constant predictor.

plot(variogram(log(zinc) ~ 1, meuse))

The next script fits a variogram model using the sample data. The function vgm() generates this model and their argument Sph indicates the selected function, a spherical model.

vgm1 = variogram(log(zinc) ~ 1, meuse)
model.1 = fit.variogram(vgm1,vgm("Sph"))
plot(vgm1, model=model.1)

Why is a variogram built and used in some interpolations? (1) The variogram tests if the semivariance of the attribute variable (response variable) increases with the separation of the points, evidencing a spatial process. (2) Variogram is the basis of various prediction and simulation algorithms, such as kriging (Li, 2022).

4.1) Simple Kriging.

Kriging also known as Gaussian process regression, is a method of interpolation based on Gaussian process governed by covariances. Kriging produces the most probably linear unbiased prediction at unsampled locations by computing a weighted average of the known attribute values of the Kriging function (lamda) in a neighborhood of the point to interpolate. The predictions of simple kriging are based on the next equation, which is a modified version of the general equation presented at the beginning of this tutorial (Li, 2022).

\[ Y(x_0) = {\sum_{i=1}^{n}{lamda_1 y(x_i)}} + [1-{\sum_{i=1}^{n}{lamda_1}}]U \]

where U is a known stationary mean, assumed to be constant over the whole domain and calculated as the average of sample data.

Since observations are considered independent, we first need to estimate the ordinary least squares residuals to estimate a kriging (regular regression residuals). We later estimate the variogram model to define the covariance structure. At the end, the function fit.variogram() fit the variogram Model to the sample Variogram to estimate the values for the parameters Nugget (it reflects the variance of sampling errors agree with the distance) and spherical model. These values are going to be included in the kriging estimation.

f = log(zinc) ~ sqrt(dist)
vt = variogram(f, meuse)
vt.fit = fit.variogram(vt, vgm("Sph"))
vt.fit
##   model      psill    range
## 1   Nug 0.07981792   0.0000
## 2   Sph 0.14905756 872.7236

Function krige allows to run a simple kriging using the values of the parameters previously estimated. Finally we plot the predicted model.

lz.sk = krige(log(zinc) ~ 1, meuse, meuse.grid, vt.fit)
## [using ordinary kriging]
meuse.grid$pred = lz.sk$var1.pred
lz.sk.raster = raster(meuse.grid["pred"])
image(lz.sk.raster,useRaster = T)
